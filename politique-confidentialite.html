Politique de confidentialité

1. Données collectées

Le site ne collecte aucune donnée automatiquement. Les seules informations recueillies sont celles envoyées volontairement via email, SMS ou WhatsApp.

2. Utilisation des données

Les données sont utilisées uniquement pour répondre aux demandes de devis ou d’information.

3. Partage des données

Aucune donnée n’est vendue ou partagée avec des tiers.

4. Durée de conservation

Les messages clients peuvent être conservés jusqu’à 3 ans pour le suivi administratif.

5. Vos droits (RGPD)

Vous pouvez demander la suppression ou la modification de vos données en écrivant à : mathieu@norouzpeinture.fr
